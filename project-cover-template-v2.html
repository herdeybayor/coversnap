Figma Recreation Guide:

Frame: 1080×1080px, radial gradient (#1a1a2e → #0a0a0f), dot grid at 3% opacity

Cropped Mobile: Phone at ~55% width, positioned from top, bottom fades into background
3D Perspective Browser: Browser at ~90% width, apply CSS transform: perspective(1200px) rotateY(-12deg) rotateX(4deg). In Figma: use "Rotate" + skew, or export from a 3D mockup tool like Rotato/Angle
Cropped Browser: Browser at ~95% width, title bar near top, page extends past bottom with gradient fade

Grayscale in Framer: CSS filter grayscale(100%) → grayscale(0%) on hover, 0.5s ease
Export: 2x (2160×2160) PNG for retina
